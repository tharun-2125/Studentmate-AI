# 🤖 StudentMate AI
# live demo
https://studentmate-ai-e8pyw6rgjo4f3yz5kpnqhx.streamlit.app/
🚀 **StudentMate AI** is an intelligent assistant designed specially for **school and college students** to help with:

- 🧠 Project Ideas
- 🏆 Hackathon Guidance
- 📊 PPT Generation
- 📚 Summarizing Tough Questions
- 💡 Tech Stack Suggestions
- 🎤 Pitch Preparation

It provides **smart, short and useful answers** based on the student's level.

---

# 🌟 Features

## 🎓 College Student Mode
Designed for hackathons and technical learning.

Includes:
- 🚀 Hackathon Ideas
- 💻 Project Suggestions
- 📊 PPT Generator
- 🎤 Pitch Creator
- ⭐ Idea Scoring
- 🧠 Tech Stack Suggestions

---

## 🏫 School Student Mode
Designed for beginners and school-level learning.

Includes:
- 📚 Small Project Ideas
- 📝 Summary Generator
- ❓ Simple Explanations
- ✏️ Short Answers
- 🧪 Science Activities
